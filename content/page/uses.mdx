## Desk

- **[Mac Mini (2020)](https://www.apple.com/mac-mini/)**: The M1 chip is just amazing.
- **[Dell UltraSharp 32" 4K Monitor](https://www.amazon.se/U3219Q-sk%C3%A4rm-DisplayPort-reaktionstid-svart/dp/B07H5MR6GD)**: I love being able to charge my computer via USB-C.
- **[Permobil F5 Corpus](https://www.permobil.com/en-us/products/power-wheelchairs/permobil-f5-corpus)**: I sit all day, so I might as well get a good chair.

## Keyboard

- **[Apple Magic Keyboard](https://www.apple.com/shop/product/MK2A3LL/A/magic-keyboard-us-english)**: My preferred wireless keyboard
- **[Logitech MX Master 3S](https://www.logitech.com/en-us/products/mice/mx-master-3s-mac-bluetooth-mouse.910-006570.html)**: Can't live without this mouse.

## Coding

- **[VSCode](https://code.visualstudio.com)**: After using Sublime for many years, I moved to VSCode like everybody else.
- **[Fira Code](https://github.com/tonsky/FiraCode)**: I absolutely love this monospaced font because of all the ligatures.

## Terminal

- **[Warp](https://www.warp.dev/)**: Rust-based terminal with AI built in.
- **[Starship](https://starship.rs/)**: The minimal, blazing-fast, and infinitely customizable prompt for any shell!
- **[Neovim](https://neovim.io/)**: Neovim is a fork of Vim aiming to improve user experience, plugins, and GUIs.

## Apps

- **[LastPass](https://LastPass.com)**: The first thing I install on any new device.
- **[Bartender](https://www.macbartender.com/Bartender4)**: Perfect way to declutter and manage the macOS menubar.
- **[CleanShot X](https://cleanshot.com)**: This is what I use whenever I need to share and/or highlight screenshots.
- **[Divvy](https://mizage.com/divvy)**: Tiny app that I use to create custom window positions.
- **[Figma](https://figma.com)**: I never thought something would replace the Adobe suite for me. Figma did.
- **[iStat Menu](https://bjango.com/mac/istatmenus/)**: A little system monitoring tool that sits on my menubar.
- **[Linear](https://linear.app/)**: A totally gorgeous issue tracker. Jira no more.
- **[Notion](https://www.notion.so/)**: My personal OS with all the notes from all my projects.
- **[TickTick](https://ticktick.com/)**: My current choice for to-do lists and organizing personal tasks.

## Services

- **[Vercel](https://vercel.com)**: Here is where I host all my websites. By far the best developer experience.
- **[Cloudflare](https://www.cloudflare.com)**: The DNS service I use with all my domains. Amazing product.
- **[Firebase](https://firebase.google.com)**: A no-brainer database and auth solution for side projects.
- **[Google Domains](https://domains.google)**: I buy and renew all my domains here.
- **[Infisical](https://infisical.com)**: An open-source secrets manager for developers.
- **[Resend](https://resend.com)**: The new email API for developers.

## Reading

- **[Kindle Paperwhite](https://amzn.to/3mAbMmm)**: I love reading on my Kindle. It's the best way to disconnect.
- **[Audible](https://www.audible.com/)**: The perfect choice to listen to a book while running outside.
